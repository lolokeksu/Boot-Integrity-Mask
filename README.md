# Boot-Integrity-Mask
Модуль скрывает модификацию загрузочных разделов (boot, init_boot, vendor_boot, vbmeta, recovery, dtbo) и динамических разделов (super) от приложений, читающих их через /dev/block/by-name. Подмена включается на раннем этапе загрузки и поддерживается адаптивным watchdog’ом всё время работы устройства.
